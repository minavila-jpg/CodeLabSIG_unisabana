## 📌 Descripción del Proyecto

Este repositorio fue desarrollado en el marco de la asignatura **Sistemas de Información Gerencial**.

El objetivo no fue únicamente programar, sino comprender cómo funcionan los entornos de desarrollo en la nube, la estructuración lógica del código y los procesos de transformación de datos bajo una metodología similar a la utilizada en la industria tecnológica.

Este proyecto integra:

- Google Colab como entorno de desarrollo en la nube  
- GitHub para control de versiones  
- Python para procesamiento y transformación de datos  
- Interpretación de imágenes mediante el modelo RGB  
- Generación de archivos en formato XLSX  

---

## 🚀 Descripción del Proceso

El ejercicio siguió un flujo estructurado de desarrollo:

1. Partir de una imagen en formato `.JPG`  
2. Convertir la imagen en una matriz de valores RGB  
3. Exportar esa matriz a un archivo `.XLSX`  
4. Reinterpretar la matriz RGB  
5. Generar un nuevo `.XLSX` con la reconstrucción correcta de los colores  
6. Repetir todo el proceso utilizando una imagen propia  

Este flujo refleja una metodología real de desarrollo:

✔ Planear  
✔ Analizar  
✔ Desarrollar  
✔ Probar  
✔ Ajustar  
✔ Versionar  

---

## 🧠 Conceptos Clave Aplicados

### 📥 Input (Entrada) vs 📤 Output (Salida)

- **Input:** Imagen en formato JPG o matriz RGB en XLSX  
- **Procesamiento:** Descomposición y reinterpretación de valores RGB  
- **Output:** Archivo XLSX con la reconstrucción correcta de los colores  

---

### 🖼️ Modelo RGB

- Una imagen está compuesta por píxeles  
- Cada píxel contiene tres valores: Rojo (Red), Verde (Green) y Azul (Blue)  
- Estos valores pueden organizarse en una matriz de datos  
- La matriz puede exportarse, manipularse y reconstruirse  

Este ejercicio demuestra cómo una imagen puede convertirse en datos estructurados y cómo esos datos pueden reinterpretarse correctamente.
